# VJ Market Dashboard

![Deployment Status](https://github.com/KVKUMARSG/VIJAY-MARKET-DASHBOARD-GAG/actions/workflows/deploy.yml/badge.svg)

A comprehensive market analysis dashboard built with React, TypeScript, and Tailwind CSS.

## Features
- Real-time stock market data (Nifty 50)
- Cryptocurrency tracking
- Currency exchange rates
- Portfolio management
- News aggregation
